Política de Privacidade

Kontrol

Última atualização: 28/05/2026

Esta Política de Privacidade descreve como o Kontrol coleta, utiliza, armazena e protege dados pessoais no uso do aplicativo, em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD - Lei nº 13.709/2018).

Ao utilizar o aplicativo, o usuário declara estar ciente das práticas descritas nesta política.

1. Dados Tratados

O Kontrol pode tratar nome, e-mail, informações de autenticação, foto ou avatar de perfil, dados financeiros cadastrados pelo usuário, preferências do aplicativo e dados técnicos necessários ao funcionamento de sessão, notificações locais e widget do dispositivo.

Os dados financeiros podem incluir contas, categorias, transações, valores, datas, observações, orçamentos, metas, relatórios e configurações de visualização.

As senhas são processadas pelo serviço de autenticação utilizado pelo aplicativo e não são armazenadas diretamente pelo desenvolvedor.

2. Finalidades

Os dados são utilizados para criar e gerenciar contas, autenticar usuários, registrar e organizar informações financeiras, exibir saldos, relatórios, gráficos, metas e análises, sincronizar dados com a infraestrutura de backend, manter preferências, proteger o serviço e viabilizar a exclusão de conta.

O aplicativo também pode usar dados para gerar relatórios em PDF, compartilhar exportações quando solicitado pelo usuário, atualizar widgets locais, solicitar permissões de galeria para avatar e agendar notificações locais relacionadas ao uso do app.

3. Serviços de Terceiros

O Kontrol não vende dados pessoais. Alguns dados podem ser processados por serviços necessários ao funcionamento do aplicativo, incluindo Supabase, para autenticação, banco de dados, armazenamento e funções de backend; e Google, para login com conta Google quando escolhido pelo usuário.

Recursos do próprio dispositivo e do sistema operacional podem ser utilizados para notificações locais, seleção de imagens, compartilhamento de arquivos, widgets e geração/abertura de documentos.

Esses serviços possuem suas próprias políticas de privacidade e medidas de segurança.

4. Armazenamento Local, Exportação e Compartilhamento

O Kontrol pode manter no dispositivo dados necessários à sessão autenticada e preferências locais, como tema e estado de visualização. Dados exibidos em widgets ficam restritos ao ambiente local do dispositivo.

Quando o usuário solicita exportação, compartilhamento ou relatório em PDF, as informações selecionadas podem ser encaminhadas para os aplicativos ou serviços escolhidos pelo próprio usuário no fluxo de compartilhamento do sistema operacional.

5. Segurança e Retenção

Os dados são armazenados em serviços de backend utilizados pelo aplicativo, especialmente o Supabase. São adotadas medidas técnicas razoáveis para proteger os dados contra acesso não autorizado, alteração, divulgação ou destruição.

Os dados são mantidos enquanto a conta estiver ativa ou enquanto forem necessários para as finalidades descritas nesta política, observadas obrigações legais aplicáveis.

6. Exclusão de Conta

O usuário pode excluir sua conta pelo próprio aplicativo. A exclusão remove os dados associados à conta, incluindo transações, contas, categorias, configurações e avatar, conforme o fluxo implementado no Kontrol.

7. Direitos do Usuário

Nos termos da LGPD, o usuário pode solicitar confirmação de tratamento, acesso, correção, anonimização, bloqueio, eliminação, portabilidade, informação sobre compartilhamento e revogação de consentimento, quando aplicável.

8. Crianças e Adolescentes

O Kontrol não é destinado a crianças. Caso seja identificado tratamento de dados de menor de idade sem consentimento adequado, os dados poderão ser removidos mediante solicitação.

9. Alterações

Esta Política de Privacidade poderá ser atualizada periodicamente. A versão mais recente ficará disponível na página pública da política.
